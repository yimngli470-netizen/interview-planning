Senior DP I — Rolling Array & Game DP

Two upgrades to basic DP: the rolling array, which collapses the space when a state only depends on the last row or two; and game / memoized-search DP, where you reason about the opponent's best play.

Source · “Senior Dynamic Programming I” deck + lecture Version · Advanced Ch5 Language · Python templates
Recap

The four elements, still

Every DP is State · Function · Initialization · Answer, and the state computes one of three things: an optimum (max/min), a yes/no, or a count. This chapter keeps that recipe and adds two optimizations/twists on top.

Optimization

Rolling array

If f[i] depends only on a fixed window of recent states, you don't need the whole table — reuse slots with modular indexing. The modulus equals how many prior states you must keep.

OriginalRollingSpace
f[i] = max(f[i-1], f[i-2]+A[i])f[i%2] = …f[(i-1)%2]…f[(i-2)%2]O(n) → O(1)
f[i][j] uses only row i-1f[i%2][j] uses f[(i-1)%2][j]O(mn) → O(n)

The trick: look at which states you actually retain, and let the modulus match.

Sequence DP

House Robber I & II

State f[i] = most you can rob from the first i houses. At house i you either skip it (f[i-1]) or rob it and skip its neighbor (f[i-2] + A[i-1]). Only two prior states matter → roll down to O(1) space:

Python · House Robber (rolled to two variables) 
def rob(nums):
    prev2, prev1 = 0, 0             # f[i-2], f[i-1]
    for x in nums:
        prev2, prev1 = prev1, max(prev1, prev2 + x)
    return prev1
House Robber II — the circular twist The houses form a circle, so the first and last are adjacent and can't both be robbed. Reduce to the line version twice: rob nums[:-1] (skip the last) or nums[1:] (skip the first), and take the better: 
def rob_circular(nums):
    if len(nums) == 1:
        return nums[0]
    return max(rob(nums[:-1]), rob(nums[1:]))
2-D coordinate DP

Maximal Square

Largest all-ones square in a 0/1 matrix. State f[i][j] = the side of the biggest square whose bottom-right corner is (i, j). If the cell is 1, it extends the smallest of its three neighbors:

Python 
def maximal_square(matrix):
    if not matrix or not matrix[0]:
        return 0
    m, n = len(matrix), len(matrix[0])
    f = [[0] * n for _ in range(m)]
    best = 0
    for i in range(m):
        for j in range(n):
            if matrix[i][j] == 1:
                f[i][j] = 1 if i == 0 or j == 0 \
                          else min(f[i-1][j], f[i][j-1], f[i-1][j-1]) + 1
                best = max(best, f[i][j])
    return best * best                    # area = side²
Grid addressing tip Locate squares by their bottom-right corner; locate rectangles by top-left + bottom-right. Since f[i][j] needs only row i-1, you can roll it to f[i%2][j] for O(n) space.
Memoized search · game theory

Coin in a Line (game DP)

Two players alternate; you must reason about the opponent playing optimally. The mindset: I win if I can hand the opponent a losing position.

Coin in a Line I — n coins, take 1 or 2 each turn, whoever takes the last coin wins. State f[i] = does the player to move win with i coins left?

Python 
def first_will_win(n):
    if n == 0:
        return False
    f = [False] * (n + 1)
    f[1] = f[2] = True
    for i in range(3, n + 1):
        # win if either move leaves the opponent in a losing state
        f[i] = (not f[i - 1]) or (not f[i - 2])
    return f[n]                            # (first player loses ⇔ n % 3 == 0)
II and III Coin in a Line II — coins have values, take 1 or 2 from the front, maximize your total: f[i] = max(A[i] + min(f[i+2], f[i+3]), A[i]+A[i+1] + min(f[i+3], f[i+4])) (the min is the opponent minimizing your future). Coin in a Line III — take from either end: an interval DP f[i][j], which leads straight into the next chapter. Also in this chapter: Longest Increasing Subsequence as memoized search.
Conclusion

The Senior DP I checklist

  1. Rolling array: when a state depends only on the last k rows, index by i % k to shrink space (O(n)→O(1), O(mn)→O(n)).
  2. House Robber = sequence DP f[i]=max(f[i-1], f[i-2]+A[i]); the circular II solves the line twice (drop first or last).
  3. Maximal Square: address by bottom-right corner; f=min(3 neighbors)+1.
  4. Game DP: I win iff I can move the opponent into a losing state; the opponent's turn is a min over my futures.
  5. Take-from-both-ends games become interval DP — the bridge to Senior DP II.
