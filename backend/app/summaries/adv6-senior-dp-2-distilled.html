Senior DP II — Interval, Matching & Knapsack

The three “heavy” DP families that show up in hard interviews: interval DP (think about the last operation, big-to-small), matching DP on two sequences (fill a grid), and knapsack DP (capacity as a dimension).

Family 1

Interval DP

Signature: the answer is over a range, you build up by merging sub-intervals, and the final answer is the whole interval [0, n-1]. The trap (“dead end”) is enumerating the first operation; the unlock is reverse thinking — reason about the last merge/burst (big-to-small, like divide & conquer).

Stone Game

· min cost to merge all stones

State dp[i][j] = min cost to merge stones i…j. Split at the last merge point k: dp[i][j] = min(dp[i][k] + dp[k+1][j]) + sum(i…j).

Python 
def stone_game(A):
    n = len(A)
    if n <= 1:
        return 0
    prefix = [0] * (n + 1)
    for i in range(n):
        prefix[i + 1] = prefix[i] + A[i]
    dp = [[0] * n for _ in range(n)]
    for length in range(2, n + 1):             # grow the interval
        for i in range(n - length + 1):
            j = i + length - 1
            dp[i][j] = min(dp[i][k] + dp[k + 1][j] for k in range(i, j)) \
                       + (prefix[j + 1] - prefix[i])
    return dp[0][n - 1]

Burst Balloons

· max coins, last-to-burst trick

Pad with 1s on both ends. State dp[i][j] = max coins from bursting all balloons strictly between i and j. Pick k as the last balloon burst in that gap — then its neighbors are exactly the boundaries i and j:

Python 
def burst_balloons(nums):
    arr = [1] + nums + [1]
    n = len(arr)
    dp = [[0] * n for _ in range(n)]
    for length in range(2, n):                  # gap between boundaries
        for i in range(n - length):
            j = i + length
            for k in range(i + 1, j):           # k = last balloon burst
                dp[i][j] = max(dp[i][j],
                               dp[i][k] + dp[k][j] + arr[i] * arr[k] * arr[j])
    return dp[0][n - 1]
The interval-DP mindset Don't ask “what happens first?” — ask “what happens last?” Fixing the last operation cleanly splits the interval into two independent sub-intervals. Also includes Coin in a Line III and Scramble String (a 3-D interval DP).
Family 2

Matching (two-sequence) DP

State f[i][j] = an answer for the first i chars of sequence A against the first j chars of B. Transition studies the relationship between char i and char j. Technique: draw the matrix and fill it.

Python · Longest Common Subsequence (max) 
def longest_common_subsequence(A, B):
    n, m = len(A), len(B)
    f = [[0] * (m + 1) for _ in range(n + 1)]
    for i in range(1, n + 1):
        for j in range(1, m + 1):
            if A[i - 1] == B[j - 1]:          # the i-th char of A is A[i-1]
                f[i][j] = f[i - 1][j - 1] + 1
            else:
                f[i][j] = max(f[i - 1][j], f[i][j - 1])
    return f[n][m]
Python · Edit Distance (min) 
def edit_distance(word1, word2):
    n, m = len(word1), len(word2)
    f = [[0] * (m + 1) for _ in range(n + 1)]
    for i in range(n + 1):
        f[i][0] = i                          # delete all i chars
    for j in range(m + 1):
        f[0][j] = j                          # insert all j chars
    for i in range(1, n + 1):
        for j in range(1, m + 1):
            if word1[i - 1] == word2[j - 1]:
                f[i][j] = f[i - 1][j - 1]
            else:
                f[i][j] = 1 + min(f[i - 1][j],      # delete
                                  f[i][j - 1],      # insert
                                  f[i - 1][j - 1])  # replace
    return f[n][m]

Same template: K Edit Distance, Distinct Subsequence, Interleaving String (and Longest Common Substring, which must be contiguous — reset to 0 on mismatch).

Family 3

Knapsack DP

The capacity becomes a dimension of the state. Backpack I: can items fill a capacity (yes/no, or max filled weight)? Backpack II: maximize value within capacity. The 1-D rolling form iterates capacity backward so each item is used once.

Python · Backpack II (max value, 0/1) 
def backpack_ii(capacity, weights, values):
    f = [0] * (capacity + 1)             # f[w] = best value using capacity w
    for i in range(len(weights)):
        for w in range(capacity, weights[i] - 1, -1):   # backward ⇒ item used once
            f[w] = max(f[w], f[w - weights[i]] + values[i])
    return f[capacity]
Family members Backpack I (feasible fill), Backpack II (max value), K Sum (count subsets of size k summing to a target — add a count dimension), Minimum Adjustment Cost (value range as a dimension). Iterate capacity forward instead for unbounded knapsack (item reusable).
Conclusion

The Senior DP II checklist

  1. Interval DP: state dp[i][j] over a range; reason about the last operation; grow intervals by length (Stone Game, Burst Balloons, Scramble String).
  2. Matching DP: f[i][j] over prefixes of two sequences; on a match take the diagonal, else combine neighbors (LCS max, Edit Distance min). Draw & fill the grid.
  3. Remember the i-th char is index i−1 in 1-indexed states.
  4. Knapsack DP: capacity is a state dimension; 1-D rolling iterates capacity backward (0/1) or forward (unbounded).
  5. Hard DP almost always reduces to one of these three shapes.
